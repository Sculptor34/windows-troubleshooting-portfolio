# Printer OKI Microline 5790 unable to print 

# Overview
This project demonstrates the how to solve printer issue.

## Problem
The PC displays device manager error.

# Diagnosis
- The printer driver unable to print test page.
- The driver display error logo

# Solution
Steps performed:
1. Remove the printer driver
2. Scan and troubleshoot the driver printer
3. Reset the physical printer
4. Restart the PC

# Commands Used
N/A

# Result
The Printer able to print test page
