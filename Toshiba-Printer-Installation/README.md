# How to connect Toshiba Printer BX410 203 dpi  in Windows 

# Overview
This project demonstrates about Toshiba Printer Installation

## Problem
Request connect Toshiba Printer via network or USB cable.

# Diagnosis
Toshiba Printer unable to communicate with PC.

# Solution
Steps performed:
1. Install the Toshiba Printer Driver
2. Add the IP printer in Driver (via network)
3. Add the Printer Driver in Printer & Scanner (Via USB cable)
4. Set up the Toshiba Printer settings
5. Edit printer preferences in PC
6. Print test page

# Result
Toshiba Printer able to print the label sticker
