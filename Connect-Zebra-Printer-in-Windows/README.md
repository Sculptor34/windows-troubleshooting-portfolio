# How to connect Zebra ZDesigner ZT410-203dpi ZPL in Windows 

# Overview
This project demonstrates how to connect Zebra printer in Windows 10

# Problem
Zebra Printer failed to communicate with PC

# Diagnosis
- PC unable to communicate with Zebra printer
- Zebra printer displays ribbon out error

# Solution
Steps:
1. Install the Zebra Printer Driver
2. Remove the Zebra Printer in Control Panel after finish install Driver
3. Add the Printer via select share printer by name
4. Click on the printer name in select share printer by name
5. Calibrate the printer Zebra as it displays ribbon out error
6. Printer able to test print page

# Commands Used
N/A

# Result
Printer able to communicate with Zebra Printer

# Notes
This method should only be used for authorized system recovery.
