# CMOS/Bios checksum error

# Overview
This project demonstrates the how to solve CMOS checksum error.

## Problem
The windows unable to boot up as normal startup.

# Diagnosis
- The PC unable to reboot.

# Solution
Steps performed:
1. Change the CMOS battery
2. Select load optimized defaults then boot
3. Synchronize the clock based on time.windows

# Commands Used
N/A

# Result
The PC able to boot up as normal startup
