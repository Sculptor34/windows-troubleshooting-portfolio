# How to connect Zebra ZDesigner ZT410-203dpi ZPL in Windows 

# Overview
This project demonstrates about Zebra Printer Installation

# Problem
Request connect Zebra Printer via network 

# Diagnosis
- Zebra Printer unable to communicate with PC 

# Solution
Steps:
1. Install the Zebra Printer Driver
2. Add the IP printer in Driver
3. Calibrate Zebra Printer
4. Edit printer preferences

# Commands Used
N/A

# Result
Zebra Printer able to print the label sticker

# Notes
This method should only be used for authorized system recovery.
